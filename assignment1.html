Headings

Headings are created by using the <h1> to <h6> tags. For example 
 
        <h1> This is a LARGE HEADING </h1> 
        <h2> This is a Large heading </h2>
        <h6> this is the smallest smallest heading </h6>
when evaluated becomes:

This is a LARGE HEADING

This is a Large heading

this is the smallest smallest heading

Text

Text is just written within the <body> tags, but can be encoded in paragraphs inside <p> tags. Bold text is written by the <b> tag. So, for example: 

        <p> This is a paragraph </p>
        <p> <b> This is another paragraph </b> </p>
when evaluated becomes:

This is a paragraph

This is another paragraph

Images

Images are added by using the <img> tag. The syntax is as follows: 

        <img src="url" alt="some_text" height="42" width="42">

The img tag is one that takes multiple arguments. Arguments are independent variables that the user specifies to define the output of the function. For example, the src argument controls the source of the image file to display, often a url. Instead of a url, the src can also be the path to an image file that is relative to the html file. If the image is in the same directory as the html file, it can just be the filename. the alt argument specifies the text that is to be displayed if the image can't be loaded for some reason (for example if the filepath specified for src is invalid). The arguments height and width control (surprise...) the height and width of the image, but are not necessary if you are happy with the size of the image as-is.
