Assignment 1: Static Webpage

For your next assignment, you will be creating a basic static webpage with header, text, and image elements to get practice using html tags. It will also help you get practice using Vim and git.

Headings

Headings are created by using the <h1> to <h6> tags. For example 
 
        <h1> This is a LARGE HEADING </h1> 
        <h2> This is a Large heading </h2>
        <h6> this is the smallest smallest heading </h6>
when evaluated becomes:

This is a LARGE HEADING

This is a Large heading

this is the smallest smallest heading

Text

Text is just written within the <body> tags, but can be encoded in paragraphs inside <p> tags. Bold text is written by the <b> tag. So, for example: 

        <p> This is a paragraph </p>
        <p> <b> This is another paragraph </b> </p>
when evaluated becomes:

This is a paragraph

This is another paragraph

Images

Images are added by using the <img> tag. The syntax is as follows: 

        <img src="url" alt="some_text" style="width:width;height:height;">
